Alla verktyg du behöver i en enda app

Smart jobbsök

Samlar jobb från flera källor på ett ställe. Med stöd för Arbetsförmedlingens API och utbyggbart för fler källor.

AI-genererade brev

Skapa skräddarsydda personliga brev baserat på jobbbeskrivning och din profil. Välj mellan olika stilar.

CV-design i många stilar

Från professionella till kreativa teman. Appen kan generera CV:n som passar just ditt yrke och personlighet.

Direktansökan via e-post

Förifyllda e-postutkast – fyller automatiskt i mottagare, ämne och bifogar dina dokument så du bara behöver granska och skicka..

Flexibel AI-integration

Stöd för både lokala AI-modeller (för integritet) och moln-API:er (för snabbhet).

Favoritlista och uppföljning

Håll koll på alla jobb du är intresserad av. Se vilka du ansökt till och följ upp.

Om projektet

Tekniken bakom MyCareerKit

Byggt för effektivitet och flexibilitet

MyCareerKit är ett skräddarsytt verktyg för att förenkla jobbsökandet. Istället för att hoppa mellan tio olika sidor och program samlas allt på ett ställe.

Appen är byggd med Python och Qt för ett snabbt och stabil gränssnitt som fungerar på både Windows och Mac.

Python Qt/PySide6 Arbetsförmedlingens API LLM-integration Gmail API HTML/CSS-generering
